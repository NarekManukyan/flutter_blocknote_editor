# BlockNote Editor Example

This example demonstrates how to use the Flutter BlockNote Editor package.

## Features Demonstrated

- Basic editor initialization
- Transaction handling
- Read-only mode toggle
- Editor state management
- Transaction logging

## Running the Example

```bash
cd example
flutter run
```

## Usage

1. The editor will initialize with an empty document
2. Type in the editor to see transactions being emitted
3. Use the lock icon to toggle read-only mode
4. Use the refresh icon to reset the editor
5. View transaction logs in the bottom panel
